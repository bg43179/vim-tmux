# zsh vimrc setting


git clone https://github.com/michaelchen110/Environment.git

Linux && Docker 
```
cd Environment

```

Mac
```
brew install zsh 
chsh -s $(which zsh)
brew install vim --with-lua
```

```
# clone

# install
cd fonts

# clean-up a bit
cd ..

```
